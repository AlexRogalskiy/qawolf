<Header>Use localhost</Header>

In this tutorial, we'll create a test on `localhost`. For now, you should use a service like <ExternalLink href="https://ngrok.com/">ngrok</ExternalLink> or <ExternalLink href="https://github.com/localtunnel/localtunnel">localtunnel</ExternalLink> to do this.

This is helpful for:

- testing a new feature while you are developing it
- adding test attributes like `data-qa=search-button` to your site while creating a test

We are considering making testing `localhost` easier. If this is something you're interested in, please <ExternalLink href="https://slack.qawolf.com">contact us</ExternalLink>.

<SubHeader>Start localtunnel</SubHeader>

First, we need to install a service like <ExternalLink href="https://ngrok.com/">ngrok</ExternalLink> or <ExternalLink href="https://github.com/localtunnel/localtunnel">localtunnel</ExternalLink>. Install localtunnel with the following command:

```bash
npm install -g localtunnel
```

First we need to start a local runner.

Now create a URL to your site by running the following command. This will create a URL that points to `localhost:3000`:

```bash
lt --port 3000
```

The result of this command is a URL you can use to create your test.

<Image
  alt="Localtunnel URL"
  height={86}
  src="/docs/use-localhost/localtunnel-url.png"
  width={524}
/>

To stop running localtunnel, press `CTRL+C`.

<SubHeader>Test localhost</SubHeader>

To create a test on `localhost`, use the tunnel URL in your test.

```js
const { context } = await launch();
const page = await context.newPage();
await page.goto("https://dry-sloth-7.loca.lt");
```
