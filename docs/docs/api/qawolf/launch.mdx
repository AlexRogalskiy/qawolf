Launch a [Playwright `Browser`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browser) instance.

### Arguments

<ArgumentLaunch />

### Returns

<ReturnBrowser />

### Examples

Launch a [Chromium](https://www.chromium.org/Home) browser with the [default options](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsertypelaunchoptions):

```js
const browser = await launch();
```

Launch a [Firefox](https://www.mozilla.org/en-US/firefox/new) browser with the [default options](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsertypelaunchoptions):

```js
const firefoxBrowser = await launch({ browserName: 'firefox' });
```

Launch a [Firefox](https://www.mozilla.org/en-US/firefox/new) browser, setting the [`slowMo` option](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsertypelaunchoptions) to `500` milliseconds:

```js
const firefoxBrowser = await launch({ browserName: 'firefox', slowMo: 500 });
```
