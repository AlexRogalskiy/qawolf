Find and click an element. If the element is not found before timeout, an error is thrown.

### Arguments

<ArgumentSelector />
<ArgumentFindElementOptions />

### Returns

<ReturnElementHandle />

## Examples

### CSS Selectors

Click on the element with the id `"my-id"`:

```js
await browser.click({ css: "#my-id" });
```

Click on the element where the `data-qa` attribute is `"submit"`:

```js
await browser.click({ css: "[data-qa='submit']" });
```

### HTML Selectors

Find and click on an element using the [default selector logic](../../use_custom_selectors#default-selector-logic):

```js
await browser.click(selectors[0]);
```

### Text Selectors

Click on the element containing the text `"Submit"`:

```js
await browser.click({ text: "Submit" });
```

### Options

Click on the element containing the text `"Sign In"`, waiting 5000 milliseconds after it is found before clicking:

```js
await browser.click({ text: "Sign In" }, { sleepMs: 5000 });
```

Click on the element with the CSS class `"my-class"`, using [Playwright's `Page.click` method](TODOFIXLINK):

```js
await browser.click({ css: ".my-class" }, { simulate: false });
```

Return the [Playwright `ElementHandle`](https://github.com/microsoft/playwright/blob/v0.10.0/docs/api.md#class-elementhandle) instance after clicking:

```js
const element = await browser.click(selectors[0]);
```
